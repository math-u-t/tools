# hashMaster - 統合ハッシュ・パスワード・テキストツール

## 📖 概要

**3つのパワフルなツールを1つのアプリで統合**したマルチツールです。ハッシュ生成、セキュアなパスワード生成、ランダムテキスト生成がタブインターフェースで簡単に切り替えられます。

すべての処理はブラウザ内で実行されるため、データはサーバーに送信されません。

**アクセス**: https://math-u-t.github.io/tools/hashMaster/

---

## ✨ 機能

### 🔐 ハッシュミキサー

テキストを複数のハッシュ形式に変換します。

**対応形式**:
- **SHA256**: 256ビットのセキュアハッシュ
- **SHA512**: 512ビットのセキュアハッシュ
- **Base64**: テキストをBase64形式にエンコード

**特徴**:
- リアルタイム変換
- ワンクリックコピー
- コピー成功時のビジュアルフィードバック

### 🔑 パスワード生成

安全で強力なパスワードを自動生成します。

**カスタマイズ可能**:
- パスワード長：8～64文字
- 文字タイプ：大文字、小文字、数字、記号（複数選択可）
- リアルタイム強度表示

**強度メーター**:
- 🔴 弱い：セキュリティが不十分
- 🟡 中程度：バランスの取れた設定
- 🟢 強い：高いセキュリティレベル

### 🎲 ランダムテキスト

カスタマイズ可能なランダムテキストを生成します。

**テキストタイプ**:
- 英数字（A-Z, a-z, 0-9）
- アルファベット（A-Z, a-z）
- 小文字（a-z）
- 大文字（A-Z）
- 数字（0-9）
- 16進数（0-9, A-F）
- カスタム文字セット

**設定オプション**:
- テキスト長：1～500文字
- 行数：1～50行
- 大文字変換オプション

## 🎯 使用方法

### タブの切り替え

上部のタブボタンをクリックして3つのツール間を切り替えます：
- 🔐 ハッシュミキサー
- 🔑 パスワード生成
- 🎲 ランダムテキスト

### ハッシュミキサー

1. テキストを入力欄に入力
2. 自動的にSHA256・SHA512・Base64に変換
3. 各ハッシュ値の「コピー」ボタンでクリップボードにコピー

### パスワード生成

1. パスワード長をスライダーで設定（8-64文字）
2. 必要な文字タイプをチェック
3. 「パスワード生成」ボタンをクリック
4. 生成されたパスワードを「コピー」

### ランダムテキスト

1. テキストタイプを選択
2. 長さと行数を設定
3. 「テキスト生成」をクリック
4. 「コピー」ボタンでテキストをコピー

## 🔒 セキュリティ

✅ **安全な処理**:
- Web Crypto APIで安全なハッシュ生成
- `crypto.getRandomValues()`で暗号的に安全な乱数生成
- すべての処理はブラウザで実行
- データはサーバーに送信されない

⚠️ **注意事項**:
- ハッシュは一方向変換（復号不可）
- Base64はエンコードであり暗号化ではない
- パスワードはコピー後、パスワードマネージャーに保存を推奨
