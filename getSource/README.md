# getSource - HTMLソース表示ツール

## 📖 概要

指定したURLのHTMLソースを`view-source:`プロトコルで新しいタブに表示するシンプルで高速なツールです。Web開発やページ分析に最適です。

**アクセス**: https://math-u-t.github.io/tools/getSource/

---

## ✨ 機能

### 🌐 URLのソース表示
- 任意のURL（http/https）のHTMLソースを表示
- ブラウザのソースビューで表示されるため、整形・シンタックスハイライトが有効

### ⚡ 高速な起動
- 入力後、Enter キーまたはボタンをクリックで即座に表示
- 新しいタブで開くため、作業の中断がない

### ✅ URLバリデーション
- http/httpsのみに対応
- 不正なURLは拒否
- 詳細なエラーメッセージを表示

---

## 🎯 使用方法

### 基本的な使い方

1. **URLを入力**
   - テキスト入力欄にURLを入力
   - 例：`https://example.com`

2. **ソースを表示**
   - Enterキーを押す
   - または「ソースコードを表示」ボタンをクリック

3. **結果確認**
   - 新しいタブでHTMLソースが表示されます
   - ブラウザのソースビューなので、シンタックスハイライトや検索機能が使用できます

### 仕様詳細

**入力**: URLの文字列

**判定条件**:
- URLが構文的に正しい
- プロトコルが `http:` または `https:`

**振る舞い**:
- 条件を満たす場合: `view-source:<URL>` を新規タブで開く
- 満たさない場合: エラーメッセージを表示

## 🌐 技術仕様

### 使用する仕組み

- **view-source: プロトコル**: ブラウザのソースビューを開く
- **新規タブ**: 元のページを閉じず確認
- **URLバリデーション**: 標準の `URL` コンストラクタで検証

### セキュリティについて

✅ **安全な処理**:
- クライアント側のみ処理
- URLはサーバーに送信されません
- スクリプト実行なし

## ⚠️ 制限事項・注意点

### 技術的な制限

- JavaScriptでHTML ソースを直接取得することはできない（ブラウザセキュリティモデルの制限）
- CORS制約により、クロスオリジンのDOM操作は行わない
- 検索機能や補完機能は実装していない

### 確認できないコンテンツ

- **JavaScript生成HTML**: 動的に生成されたコンテンツは見えない場合がある
- **iframe の内容**: フレーム内のコンテンツは別途確認が必要
- **プライベートサイト**: 認証が必要なサイトは見る事ができない場合がある
