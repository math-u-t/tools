# 高精度 時計・タイマー・メモ（ブラウザ版）

## 概要

本ツールは、**ブラウザ標準APIのみ**を用いて実装された以下の機能を提供します。

* 高精度リアルタイム時計
* 高精度ストップウォッチ（タイマー）
* コピー操作の可視化
* 永続化される簡易メモ機能

すべて **1つの HTML ファイル**で動作し、外部ライブラリやビルド工程を必要としません。

## 機能一覧

### 1. 高精度時計

* 表示形式

  ```
  HH:MM:SS.mmm
  ```
* ミリ秒まで表示
* `Date.now()` と `performance.now()` を併用し、表示揺らぎを低減
* 表示中の時刻をワンクリックでコピー可能

### 2. 高精度タイマー（ストップウォッチ）

* 開始 / 停止 / リセット
* 表示単位：秒（小数第3位まで）
* 実装上の定義
  経過時間は

  ```
  Δt = (Date.now() - t0) + (performance.now() - p0)
  ```

  として算出し、**加算型カウンタは使用しない**
* 累積誤差が原理的に発生しない
* 表示値をコピー可能

### 3. コピー操作の可視化

* コピー成功時に以下を同時に行う

  * ボタン表示を「コピー済」に一時変更
  * 「Copied」ステータスを表示
* 表示は約 1.2 秒後に自動復帰
* `navigator.clipboard.writeText` の **成功時 Promise 解決**のみを成功と判定

### 4. メモ機能

* 自由記述テキスト
* 操作

  * 保存
  * クリア
  * コピー
* 保存先：`localStorage`
* ページ再読み込み後も内容は保持される
* **自動保存は行わない**（操作と状態遷移を一致させるため）

## 使用方法

1. 提供された HTML ファイルを保存
2. ブラウザで直接開く（ローカルで可）
3. 追加設定・インストール不要

※ コピー機能は HTTPS 環境または `localhost` での使用を推奨

## 技術仕様

| 項目| 内容|
| - | -|
| 使用言語    | HTML / CSS / JavaScript |
| 外部ライブラリ | なし|
| 時刻取得| `Date.now()`|
| 高分解能計測  | `performance.now()`     |
| 描画更新| `requestAnimationFrame` |
| 永続化| `localStorage`|

## 設計上の注意

* **計測精度 ≠ 表示更新頻度**
  表示は 60Hz 程度だが、計測値は連続実数として保持
* ミリ秒以下の精度は **表示されないが内部では保持**
* 実際の精度は OS・ブラウザ実装に依存

## 制限事項

* ナノ秒精度は理論上・実装上ともに保証されない
* ブラウザがバックグラウンドにある場合、描画頻度は低下する
* コピー機能はブラウザのセキュリティ制約に従う
