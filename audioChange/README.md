# audioChange - ブラウザ音声録音・再生ツール

## 📖 概要

**ブラウザ標準APIで直接録音・再生・管理**できる高機能な音声ツールです。Web Audio APIで波形表示、リアルタイム音量計測、複数の音声ファイル管理が可能です。

マイクへのアクセスはブラウザのセキュリティに基づき安全に処理されます。

**アクセス**: https://math-u-t.github.io/tools/audioChange/

---

## ✨ 機能

### 🎙️ 音声録音

高品質で音声を直接録音します。

**特徴**:
- **リアルタイム波形表示**: 周波数スペクトラムをカラフルに表示
- **音量レベル表示**: 実時間で音量メーターが変動
- **経過時間表示**: MM:SS形式でタイマー表示
- **ワンクリック操作**: 開始・停止・リセットが簡単

### 🎵 音声再生

保存した音声を再生します。

**再生機能**:
- **再生・一時停止**: 音声の再生制御
- **プログレスバー**: クリックで任意の位置へジャンプ
- **時間表示**: 現在位置と総時間を同時表示
- **波形表示**: 周波数スペクトラムをリアルタイム表示

### 💾 ファイル管理

複数の音声を管理・保存できます。

**ファイル操作**:
- **カスタムネーミング**: 録音ごとにファイル名を設定
- **ダウンロード**: WebM形式で保存可能
- **ローカル保存**: LocalStorageで最大複数ファイル保存
- **URL共有**: 生成されたURLをコピーして共有

### 📊 リアルタイム可視化

Web Audio APIで美しい波形表示を実現しています。

## 🎯 使用方法

### 音声録音

1. 📻 音声録音セクションで「⭕ 録音開始」をクリック
2. マイクを許可（初回のみ）
3. 波形と音量レベルがリアルタイムで表示
4. 「⏹️ 停止」をクリックで終了
5. 「🔄 リセット」で破棄（新規録音）

### 音声再生

1. 停止後、自動的に再生セクションに表示
2. ファイル名を入力（またはデフォルト値のまま）
3. 「▶️ 再生」で音声を再生
4. プログレスバーをドラッグして位置指定可能

### ファイル管理

**ダウンロード**:
1. 「⬇️ ダウンロード」をクリック
2. WebM形式で自動ダウンロード

**URL共有**:
1. 「📋 URLコピー」でURLをコピー
2. 別のタブやアプリにペースト

**ローカル保存**:
1. LocalStorageに自動保存
2. 「📁 保存済み音声」セクションに表示
3. 「再生」または「削除」操作可能

## 🔒 セキュリティとプライバシー

✅ **プライベート処理**:
- すべての処理がブラウザ内で実行
- マイク音声はサーバーに送信されない
- ローカルストレージはブラウザ内保存

✅ **マイクアクセス**:
- 最初のアクセス時のみ許可ダイアログが表示
- いつでも取り消し可能
- HTTPSまたはlocalhostで動作

⚠️ **注意**:
- ブラウザキャッシュクリアでLocalStorageが削除される可能性
- 重要な音声ファイルはダウンロードして別途保存推奨
- 共有URL有効期限なし（ブラウザがメモリに保持している間のみ）

## 🌐 技術仕様

### 使用API

| API | 説明 |
|-----|------|
| MediaRecorder API | 音声データ録音 |
| Web Audio API | 波形表示・周波数分析 |
| getUserMedia | マイクアクセス |
| Blob API | 音声データ処理 |
| localStorage | ファイル保存 |
| Canvas API | 波形描画 |

### ファイル形式

- **出力形式**: WebM（audio/webm）
- **コーデック**: Opus（標準）
- **サンプリングレート**: 48kHz（ブラウザ標準）

### パフォーマンス

- **波形更新**: 60FPS（smooth）
- **音量計測**: 100ms毎（レスポンシブ）
- **ファイル保存**: 即座（数ミリ秒）

## ⚙️ 設定・カスタマイズ

### マイクデバイス選択

複数のマイクがある場合、ブラウザのシステム設定で選択できます。

### ファイル名規則

デフォルト形式: `recording_YYYY-MM-DDTHH-mm-ss`

カスタムネーミング可能。

### LocalStorage容量

- 制限: ブラウザ通常5～10MB
- 圧縮: Base64エンコーディングで約33%容量増
- 推奨: 1～2分の音声を1ファイルとして管理

## ❓ よくある質問（FAQ）

**Q: マイクはどうやって許可しますか？**
A: 最初にアクセスした時、ブラウザのダイアログで「許可」を選択してください。

**Q: 音声ファイルはどこに保存されますか？**
A: デフォルトはブラウザのLocalStorageに保存されます。ダウンロードで自分のパソコンに保存できます。

**Q: 最大何分まで録音できますか？**
A: 理論的には無制限ですが、ブラウザのメモリ容量に依存します。通常は30分以上の録音が可能です。

**Q: 波形表示はカスタマイズできますか？**
A: 現在のバージョンでは不可ですが、コードをフォークして改造可能です。

**Q: 複数の音声を同時に再生できますか？**
A: 現在のバージョンでは1つずつの再生になります。
